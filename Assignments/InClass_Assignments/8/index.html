Create a Products table. The structure is given in the following image

QUERY USED (CODE Below):

CREATE TABLE Products (
ProductID INT(11) NOT NULL AUTO_INCREMENT,
ProductName VARCHAR(40) NOT NULL,
SupplierID INT (11) NULL,
CategoryID INT (11) NULL,
QuantityPerUnit VARCHAR(20) NULL,
UnitPrice DECIMAL (10,4) DEFAULT 0.0000,
UnitsInStock SMALLINT (2) DEFAULT 0,
UnitsOnOrder SMALLINT(2) DEFAULT 0,
ReorderLevel SMALLINT(2) DEFAULT 0,
Discontinued BIT(1) NOT NULL DEFAULT 0,
PRIMARY KEY (ProductID)
);

Question 3:

Write a query to get Product name and quantity/unit.

ANSWER SCREENSHOT:



QUERY USED (CODE Below):

SELECT ProductName, QuantityPerUnit FROM Products;

Question 4:

Write a query to get current Product list (Product ID and name)

ANSWER SCREENSHOT:




QUERY USED (CODE Below):

SELECT ProductID, ProductName
FROM Products
WHERE Discontinued = 0
ORDER BY ProductName;

Question 5:

Write a query to get discontinued Product list (Product ID and name).

ANSWER SCREENSHOT:



QUERY USED (CODE Below):

SELECT ProductID, ProductName
FROM Products
WHERE Discontinued = TRUE
ORDER BY ProductName;

Question 6:

Write a query to get most expense and least expensive Product list (name and unit price)

ANSWER SCREENSHOT:



QUERY USED (CODE Below):

SELECT ProductName, UnitPrice
FROM Products
ORDER BY UnitPrice desc;

Question 7:

Write a query to get Product list (id, name, unit price) where current products cost less than $20.

ANSWER SCREENSHOT:



QUERY USED (CODE Below):

SELECT ProductID, ProductName, UnitPrice FROM Products WHERE (((Discontinued)= 0) AND ((UnitPrice)<20)) ORDER BY UnitPrice desc;
